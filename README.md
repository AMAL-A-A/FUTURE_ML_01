# Sales & Demand Forecasting using Machine Learning

## Project Overview
This project predicts sales using historical Superstore sales data.
The model applies machine learning to forecast sales and provide business insights.


## Dataset
Superstore Sales Dataset


## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook


## Machine Learning Model
Random Forest Regressor


## Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Square Error)
- R² Score


## Visualizations
- Daily Sales Trend
- Monthly Sales
- Sales by Category
- Sales by Region
- Profit vs Sales
- Actual vs Predicted
- Error Distribution
- Feature Importance

- Regional sales vary significantly.
- Discounts impact sales.
- Office Supplies contribute consistently.
- The forecasting model helps inventory planning.
